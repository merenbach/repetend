# Repetend

Rust-based long-division calculator

## Getting started

An example run might look like this:

    $ repetend --dividend 1 --divisor 7 --scale 100
    0.1428571428571428571428571428571428571428

This can be shortened to:

    $ repetend -a 1 -b 7 -c 100
    0.1428571428571428571428571428571428571428
